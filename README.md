# 📊 Financial Dashboard

A Streamlit-based interactive dashboard for calculating **EMI (Equated Monthly Installments)** and simulating **salary revisions across different countries**.  
This app helps users compare loan scenarios and evaluate salary vs. living cost trade-offs in a clear, visual way.

---

## 🚀 Features
- **EMI Calculator**  
  - Input loan amount, interest rate, and tenure  
  - Get monthly EMI, total payment, and total interest  
  - Compare EMI across different interest rates  

- **Salary Revision Simulator**  
  - Enter current and proposed salary  
  - Input living costs for any two countries  
  - See net gains and breakeven months  
  - Compare savings growth over 5 years  

- **Interactive Charts**  
  - Bar chart: EMI comparison across interest rates  
  - Line chart: Savings growth for both countries  

---

## 🛠️ Tech Stack
- [Streamlit](https://streamlit.io/) – for building the dashboard  
- [NumPy](https://numpy.org/) – for calculations  
- [Matplotlib](https://matplotlib.org/) – for charts
